# BUG-006: Quiz Accepts Partial Answers — any() Instead of all()

## Summary
`2_Quiz.py` validated answers with `any([intro, body, conclusion])` — a user could submit with ONLY
the body section filled and the AI would grade an incomplete answer, producing misleading scores.

## Fix Applied
Changed to `all([intro_text.strip(), body_text.strip(), conc_text.strip()])`.
Updated warning message to be specific about which sections are required.
